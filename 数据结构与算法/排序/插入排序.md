# 插入排序

```java
public class InsertionSort<T extends Comparable<T>> extends Sort<T> {

    public void sort(T[] nums) {
        int n = nums.length;
        for (int i = 1; i < n; i++) {
            for (int j = i; j > 0 && less(nums[j], nums[j - 1]); j--) {
                swap(nums, j, j - 1);
            }
        }
    }

}
```
